<Accordion title="Is this a Roblox Plugin?">
  No, Nilo is a standalone platform. You create in your browser, then export assets to use in Roblox Studio.
</Accordion>

<Accordion title="What features are coming?">
  As of Q1 2026 focuses on asset creation and export—generate characters, tools, maps, optimize them, and export to Roblox. Full game creation and publishing features are coming in Q2 2026.
</Accordion>

<Accordion title="Will my assets work in existing Roblox games?">
  Yes! Nilo exports standard Roblox-compatible formats that work with any Roblox game.
</Accordion>

<Accordion title="Do I need to know Blender or 3D modeling?">
  Nope. Nilo handles all the technical optimization (retopology, rigging, texture baking) automatically. If you can describe what you want, you can create it.
</Accordion>

<Accordion title="Can I build full games in Nilo?">
  Not yet. Open Beta focuses on asset creation for Roblox. Full game creation and publishing inside Nilo is for a future release.
</Accordion>

<Accordion title="How long is Nilo in Open Beta?">
  We don't have much to share yet! Expect more information later in 2026.
</Accordion>
